f311.filetypes.basic.description_to_symbols(descr)¶

Finds molecular formula in description text and returns separate symbols

This routine is similar to pfantlib.f90:assign_symbols_by_formula()

It is used when the symbols are not informed separately after the first “#” in the molecule ‘titulo’

Parameters:descr – string
Returns:list of two elements, e.g., [” C”, ” O”]

Example:

>>> import f311.filetypes as ft
>>> ft.description_to_symbols("12C16O INFRARED  X 1 SIGMA+,   version 15/oct/98")
[' C', ' O']

f311.filetypes.basic.get_default_data_path(*args, module=None, class_=None)¶

Returns path to default data directory

Arguments ‘module’ and ‘class’ give the chance to return path relative to package other than f311.filetypes

Parameters:
  • module – Python module object. It is expected that this module has a sub-subdirectory named ‘data/default’
  • class – Python class object to extract path information from. If this argument is used, it will be expected that the class “root” package will have a subpackage called ‘filetypes’, i.e., ‘(some package).filetypes’. Has precedence over ‘module’ argument
